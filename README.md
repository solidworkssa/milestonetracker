# MilestoneTracker

Project milestone tracking with escrow payments.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/MilestoneTracker.sol`
- Stacks: `contracts/stacks/contracts/milestonetracker.clar`

## Setup
```bash
pnpm install
pnpm dev
```
